# Changelog

## 0.5.0

- Add Rule ウィザードコマンド追加（コマンドパレットから対話的にルール追加）
- 設定ページに markdownDescription で設定例を表示
- 未設定時のセットアップガイドから Add Rule を直接起動可能に

## 0.4.0

- keywords 追加（Marketplace 検索ヒット率向上）
- ルールごとのカスタム tooltip 設定対応
- `$0`〜`$n` 複数キャプチャグループ対応
- 未設定時のセットアップガイド表示

## 0.3.0

- ホバー時に遷移先ラベル一覧を tooltip 表示
- 同一位置に複数ルールがマッチした場合の重複排除
- 無効な正規表現設定時の警告メッセージ表示
- アイコンを 128x128 に最適化
- CHANGELOG.md 追加

## 0.2.0

- 説明文を汎用的な表現に更新（課題キー限定 → 任意のテキストパターン）

## 0.1.0

- 複数パターン対応: `rules` ベースの設定構造に変更
- パターンごとに異なる遷移先を設定可能に

## 0.0.1

- 初回リリース
- ターミナル上のテキストを正規表現で検出しリンク化
- QuickPick で遷移先を選択してブラウザで開く機能
